# PubMed XML to XLSX converter

pubmed_xml_to_xlsx - A simple tool to process a PubMed query XML file and store the relevant fields as XLSX.

## Getting Started

`python pubmed_xml_to_xlsx <input_file>.xml <output_file>.xlsx`
